    Nossa metodologia é embasada no princípio de que o conhecimento acontece quando o que você aprende é aplicado, portanto você viverá isso ao máximo, através da atuação em projetos reais.


Programa Starter


    Formação de Desenvolvedores Full Stack no modelo bootcamp (aprendizagem intensiva e acelerada). O programa tem duração de aproximadamente 40 semanas e tem como objetivo formar desenvolvedores com os conhecimentos e experiência necessários para desenvolver aplicações web de acordo com as melhores práticas do mercado.

Estrutura do programa Sarter


✭ 40 semanas (aproximadamente 10 meses).
✭ Aproximadamente 800 horas.
✭ Dedicação de aproximadamente 20 horas semanais. Nosso programa é puxado!
✭ Os projetos reais são realizados por times de desenvolvimento, compostos pelos alunos e nossos mentores.
✭ No programa você trabalhará:
✭ Lógica de programação, estrutura de dados e pensamento computacional.
✭ Padrões e melhores práticas para a estruturação e desenvolvimento de aplicações.
✭ Tecnologias básicas para o desenvolvimento web: HTML, CSS e JavaScript.
✭ Back-end de aplicações em Node, PostgreSQL, Redis, dentre outras tecnologias.
✭ Desenvolvimento web front-end com React.
✭ Provisionamento de infraestrutura no PaaS (platform as a service) Heroku.
✭ Processos e estratégias para desenvolvimento de features e hotfix com github.
✭ Arquitetura, funcionalidades e padrões comuns às principais aplicações de mercado.
